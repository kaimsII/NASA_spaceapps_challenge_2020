# NASA Space Apps Challenge 2020 – Ch’aska ñawi

## Descripción

Este repositorio contiene el trabajo desarrollado por el equipo **Ch’aska ñawi** para el desafío **A Flood of Ideas** del *NASA Space Apps Challenge 2020*.
El reto consistía en proponer soluciones basadas en datos de observación de la Tierra que permitan **estimar el impacto de inundaciones en infraestructura crítica y aportar metodologías útiles para la reducción de riesgos de desastre**.

## Objetivo

El proyecto, llamado **Piggybank**, busca:

* Combinar datos de múltiples satélites para monitorear, evaluar y estimar el impacto económico de inundaciones sobre infraestructura crítica.
* Diseñar un modelo que relacione variables meteorológicas con daños monetarios, permitiendo la creación de un método predictivo útil para decisiones de prevención y gestión de riesgo.

## Contenido del repositorio

El núcleo del trabajo está en:

* `NASA_apps_challenge_code.ipynb`
  Código en Python desarrollado como parte del desafío, con análisis y procesamiento de datos relevantes al problema planteado.

## Desafío: *A Flood of Ideas*

El reto pedía desarrollar **nuevas metodologías o algoritmos que aprovechen los datos de observación de la Tierra y conjuntos de datos de infraestructura crítica para estimar los daños causados por inundaciones**. La meta era contribuir a los esfuerzos de la comunidad global para reducir el riesgo de desastres utilizando datos espaciales.

## Cómo abordamos el problema

* Analizamos datos de satélites como **COPERNICUS Sentinel-1A** y **ALOS-2**, así como variables meteorológicas para establecer patrones de daño.
* El enfoque prioriza modelos que conectan eventos meteorológicos con pérdidas económicas en infraestructura para facilitar la gestión preventiva de riesgos.

## Tecnologías utilizadas

* **Python**
* **Jupyter Notebook**
* Bibliotecas de análisis de datos (NumPy, Pandas) y técnicas de procesamiento basadas en observación satelital.

## Uso de datos de observación espacial

El proyecto aprovecha datos de:

* Imágenes de satélite con radar de apertura sintética (SAR) para detectar cambios en la superficie terrestre después de eventos de inundación.
* Variables meteorológicas para correlacionar eventos climáticos con la severidad y coste estimado de daños.

## Estado del proyecto

Este trabajo fue desarrollado y entregado como parte de la hackathon *NASA Space Apps Challenge 2020*, en formato de análisis y código funcional para exploración y métricas.
